### Node.js

</div>

#### 知识点

1. 不同平台（Linux、macOS、Windows）的安装
2. 编辑器环境搭建，主要是 VSCode，包括如何与 ESLint、Prettier 等插件集成
3. nvm 工具的使用
4. 配置 npm 源，加速依赖安装

#### 实战教程

✍️ Node.js 开发环境搭建 🗝1, 2, 3, 4

> 自从 2009 年 Node.js 诞生以来，前端就呈现爆发式增长，现在火爆的前端开发技术如 React，Vue 等都专配了脚手架来进行应用的开发，
> 而脚手架背后依赖的 Webpack 等技术都需要 Node.js 的加持，并且我们平时开发 React 时候，有很多我们需要功能都是以 NPM 模块的形式发布的，
> 而我们需要安装这些模块也需要了解 Node.js 对应的内容，本篇教程致力于讲解我们平时在 React 开发过程所必备的 Node.js 知识，让你在之后 React 中如鱼得水！
